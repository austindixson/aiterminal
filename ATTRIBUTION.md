## Architecture & Design Inspiration

This project draws inspiration from several excellent tools and platforms:

- **Cursor IDE** - AI-first code editor with inline chat and agent workflows
- **Warp Terminal** - Modern terminal with AI integration and command palette
- **Claude Code** - Anthropic's CLI tool with agentic loops and tool calling
- **VSCode** - Electron application architecture, split panels, and extensibility patterns
- **Rivet** - Visual agent orchestration and node-based workflows
- **Agent Zero** - Compact agent execution model with step badges
